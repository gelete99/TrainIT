# Contribución

Gracias por tu interés en colaborar en TrainIT.

## Requisitos
- Android Studio
- JDK 11
- Node.js 22
- Firebase CLI
- Archivo `app/google-services.json` configurado en local

## Puesta en marcha

### App Android
1. Clona el repositorio.
2. Abre el proyecto en Android Studio.
3. Añade tu archivo `app/google-services.json`.
4. Sincroniza Gradle.
5. Ejecuta la app desde Android Studio.

### Firebase Functions
1. Entra en la carpeta `functions`.
2. Instala dependencias con:

```bash
npm install
npm run build
